A Little About Us

mChat is a new modernised chat website in which our clients can join rooms and interact with eachother.

We look to keep expanding our platform, its capabilites, features and its reach in order to be more user friendly and have a larger audience!

Development

This website has been developed using:

Hyper Text Markup Language (HTML5) has been used for all front-end code
Cascading StyleSheet (CSS) has been used to style the UI
JavaScript (JS) has been used for the back-end chat application
